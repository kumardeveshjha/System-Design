State patterns are used when we wanted to change the state of the system dynamically like:
  i. Google Map Direction sense 
  ii. Application where there is required different states or modes 

This helps in reducing multiple if else statements and also reduce the tight dependency between the services 
This is used to make sure the OCP is preserved 


### Best use cases:
 1. Music Player (Play pause next)
 2. E-commerce (Orderplaced, packaged, out of delivery) 
 3. ATM mahine( Idle, card inserted, pin verification, select transaction, cash withdrawal)
 4. Traffic Light system.
